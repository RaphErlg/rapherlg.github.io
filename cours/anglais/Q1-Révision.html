2.1 · Programming languages (les langages de programmation)

Why are there so many programming languages? (3 reasons) — Pourquoi y a-t-il autant de langages ?

  1. Technology is always evolving: new technologies need new tools to build their software. — La technologie évolue sans cesse : de nouvelles technos exigent de nouveaux outils (la raison principale).

  2. Different developer jobs require different languages. — Chaque métier de développeur a besoin d'outils différents.

  3. Developers have different needs and goals: speed/performance, fast development, or a specific task. — Des besoins/objectifs différents : vitesse, rapidité de dev, ou tâche précise.

❓ "There are enough programming languages, we don't need more." → FALSE. As technology keeps evolving, new problems arise that current languages can't solve, so new languages are still being created. (Faux : la techno évolue, de nouveaux problèmes apparaissent, donc on crée encore de nouveaux langages.)

🚗 The vehicles analogy — l'analogie des véhicules

Just like different vehicles (trains, tractors, bicycles) are designed for specific purposes, programming languages are created for specific tasks. → HTML for websites · Python for AI/data · C++ for video games. (Comme chaque véhicule a un usage précis, chaque langage est créé pour des tâches précises.)

🩺 The doctor-patients analogy — l'analogie médecin-patients

A doctor specializes (cardiology, dentistry…) to treat different illnesses. In the same way, there are different kinds of software, so programmers specialize just like doctors. (Un médecin se spécialise selon les maladies ; de même, les programmeurs se spécialisent.)

2.3 · ⭐ The two famous people (les 2 personnes célèbres)

👩 Ada Lovelace

  • An English mathematician of the 19th century; she studied maths around 1843, which was unusual for a woman at the time. — Mathématicienne anglaise du 19e ; étudier les maths était inhabituel pour une femme.

2.6 · Model answers (réponses-types)

✍️ To reuse / adapt. (À réutiliser / adapter.)

Analogie véhicules

Just like different vehicles are designed for specific purposes and have their own pros and cons, programming languages are created for specific tasks. For example, HTML is used for websites, while Python is preferred for artificial intelligence.

What is an OS? (3 elements)

An operating system is like a conductor: it tells the hardware what to do for the user. Three key elements are applications (run software), drivers (translators so applications can communicate with the hardware), and the kernel (the main program that manages the system's resources).

3 factors (OS choice)

First, the intended use (Windows for a desktop, Linux for a server). Second, usability: an OS must be user-friendly. Third, the type of interface: a GUI for ease of use, or a CLI because it is faster and uses fewer hardware resources.
