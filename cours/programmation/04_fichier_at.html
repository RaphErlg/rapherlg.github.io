Fichiers en C — Synthèse

Les 3 étapes obligatoires

Chaque manipulation de fichier suit toujours le même schéma :

  1. Ouvrir le fichier avec fopen

  2. Lire ou écrire avec fscanf, fgets ou fprintf

  3. Fermer le fichier avec fclose

Les types et fonctions essentielles

FILE* — la variable fichier

FILE* fichier;

Une variable spéciale qui représente le lien entre ton programme et un fichier sur le disque. Pense-y comme un signet qui pointe vers le fichier.

fopen — ouvrir un fichier

fichier = fopen("nom_du_fichier.txt", "mode");

Mode

Signification

"w"

Écriture (write) — crée le fichier ou l'écrase s'il existe

"r"

Lecture (read) — ouvre un fichier existant

Si fopen échoue (nom invalide, fichier inexistant...), il renvoie NULL.

fprintf — écrire dans un fichier

fprintf(fichier, "texte %d et %s\n", nombre, chaine);

Identique à printf mais avec le fichier comme premier argument. Tous les formats %d, %s, %lf... fonctionnent de la même façon.

fscanf — lire dans un fichier mot par mot

fscanf(fichier, "%d", &valeur);
fscanf(fichier, "%s", chaine);   // pas de & pour les chaînes
fscanf(fichier, "%lf", &prix);

Identique à scanf mais avec le fichier comme premier argument. Ignore automatiquement les espaces et retours à la ligne pour trouver la prochaine valeur.

fgets — lire une ligne entière

char texte[100];
fgets(texte, 100, fichier);

À utiliser quand le texte contient des espaces. fscanf avec %s s'arrêterait au premier espace, fgets lit toute la ligne. Attention : fgets inclut le \n de fin de ligne dans la chaîne lue.

fclose — fermer un fichier

fclose(fichier);

Toujours fermer le fichier à la fin. Sans ça, les données écrites peuvent ne pas être sauvegardées.

Le curseur de lecture

Quand un fichier est ouvert, un curseur se place au début. À chaque lecture (fscanf, fgets), il avance automatiquement jusqu'à la prochaine valeur. Tu n'as pas à gérer sa position manuellement.

3\n  7\n  2\n  8\n  1
↑
curseur au départ

-- après fscanf --

3\n  7\n  2\n  8\n  1
          ↑
          curseur après 2 lectures

Gérer l'échec de fopen

Toujours vérifier que fopen a réussi avant d'écrire ou lire :

fichier = fopen(nomFichier, "w");

if (fichier == NULL) {
    printf("Nom incorrect !\n");
} else {
    // écriture ici
    fclose(fichier);
}

Version avec redemande à l'utilisateur (boucle de validation) :

fichier = fopen(nomFichier, "w");

while (fichier == NULL) {
    printf("Nom incorrect, réessayez : ");
    scanf("%s", nomFichier);
    fichier = fopen(nomFichier, "w");
}

feof — détecter la fin du fichier

Quand on ne sait pas à l'avance combien d'éléments contient un fichier, on utilise feof :

while (!feof(fichier)) {
    fscanf(fichier, "%d", &valeur);
    printf("%d\n", valeur);
}

feof(fichier) renvoie vrai quand le curseur a buté contre la fin du fichier lors d'une lecture.

⚠️ Le piège de feof

feof ne détecte la fin de fichier que quand une lecture bute contre elle. Si le fichier se termine par un retour à la ligne :

3
7
1
        ← retour à la ligne ici

Quand fscanf lit 1, il s'arrête au \n — il n'a pas encore buté contre la fin. feof reste faux, on rentre dans la boucle une fois de plus, fscanf bute enfin contre la fin mais valeur vaut toujours 1. Résultat : 1 est affiché deux fois.

Solution : s'assurer que le fichier ne se termine pas par un retour à la ligne.

Manipuler deux fichiers simultanément

On peut avoir plusieurs variables FILE* ouvertes en même temps, chacune avec son propre curseur :

FILE* lecture  = fopen("source.txt", "r");
FILE* ecriture = fopen("resultat.html", "w");

// lire dans l'un, écrire dans l'autre
char texte[100];
fgets(texte, 100, lecture);
fprintf(ecriture, "<li>%s</li>\n", texte);

fclose(lecture);
fclose(ecriture);

Exemple complet — lecture d'un fichier de courses

Fichier courses.txt :

Pommes 3.15
Fromage 1.69
Pain 2.99

Programme :

#define _CRT_SECURE_NO_WARNINGS
#include <stdio.h>

void main(void) {
    FILE* fichier = fopen("courses.txt", "r");

    char nom[100];
    double prix;
    double total = 0;

    for (int i = 0; i < 3; i++) {
        fscanf(fichier, "%s", nom);
        fscanf(fichier, "%lf", &prix);
        printf("%s : %.2lf\n", nom, prix);
        total += prix;
    }

    printf("==================\n");
    printf("Total : %.2lf\n", total);

    fclose(fichier);
}

%.2lf dans printf affiche un double avec exactement 2 décimales.

Récapitulatif des fonctions

Fonction

Rôle

fopen(nom, mode)

Ouvre un fichier, renvoie NULL si échec

fprintf(fichier, format, ...)

Écrit dans un fichier

fscanf(fichier, format, ...)

Lit mot par mot dans un fichier

fgets(chaine, taille, fichier)

Lit une ligne entière dans un fichier

fclose(fichier)

Ferme un fichier

feof(fichier)

Renvoie vrai si la fin du fichier a été atteinte
